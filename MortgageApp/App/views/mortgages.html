Welcome

The Mortgage App is a sample web application that uses the Durandal Single-Page Application framework. Its back-end is written with ASP.NET Web API and Entity Framework.

This application demonstrates how to:

  • Setup and use Durandal from the Starter Kit for Visual Studio 2013.
  • Write a Web API controller that uses Entity Framework for simple CRUD operations.
  • Configure basic routing in Durandal.
  • Write custom Knockout binding handlers and extensions.
  • Write a rich JavaScript object model with Knockout observability.
  • Write clean, modular JavaScript code.
  • Use the Flot graph/chart library.
  • Use the KoLite command and dirty state tracking systems.
